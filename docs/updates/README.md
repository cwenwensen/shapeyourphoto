# Update Docs

`docs/updates/` 存放按版本归档的更新记录。旧版本记录不得删除；如旧记录与当前行为冲突，以最新版本文档、当前 `docs/technical/` 专题和代码为准。

## 规则

1. 每一轮正式版本升级都新增一份 `<version>.md`。
2. 更新文档应说明改了什么、为什么改、涉及文件和验证方式。
3. 旧版本文档保留历史上下文，可以追加勘误或维护说明。
4. 不得用临时 handover 或聊天摘录替代版本文档。
